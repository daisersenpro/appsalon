# AppSalon

AppSalon — Proyecto ejemplo PHP MVC (PSR-4) con Composer y ActiveRecord; frontend con SASS y Gulp para compilar assets.

## Quick start

```bash
composer dump-autoload
npm install
npm run dev   # o `npx gulp`
php -S localhost:3000 -t public
```

## Tecnologías
- PHP (MVC, PSR-4)
- Composer (autoload)
- SASS + Gulp (build frontend)
- MySQL (XAMPP)
